Product Recommender

Methodology

Review Aggregator:
Scrape web reviews from many sources. In addition to generating wordcloud visualations this part also generates feature vectors for devices to be used later in the recommender system.

Sentiment Analysis:
Runs custom built sentiment analysis model using Gradient Boosted Trees on the Reviews. The model was built based on mining over around 1.6 million electronic gadget reviews on Amazon.

User Feature Vector:
Generate user feature vector based on user inputs and previous user reviews.

Recommender System:
Recommender system to recommend products and predict a star rating for the particular user, product pair
